♟ Configuración inicial

Para que la app se sincronice entre varios usuarios necesitas un proyecto Firebase gratuito.

Cómo obtener la configuración:
1. Ve a console.firebase.google.com
2. Crea un proyecto → Agrega una app web (</>)
3. En Realtime Database → Crear base de datos → Modo de prueba
4. Copia los datos del objeto firebaseConfig aquí abajo
